# java.util.List


## length

返回值类型：int

- 获取List的长度


```
${listVar.length()}
```


## join


| 参数名 | 描述 | 可否为空 |
| --- | --- | --- |
| separator | 分割字符 | 是 |

返回值类型：String

- 将List用separator拼接在一起


```
${listVar.join(',')}
```


## shuffle

返回值类型：List

- 将list打乱顺序


```
${listVar.shuffle()}
```


## sort

返回值类型：List

- 将list排序(正序)


```
${listVar.sort()}
```
