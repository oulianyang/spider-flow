# base64


## encode


| 参数名 | 描述 | 可否为空 |
| --- | --- | --- |
| bytes | 要加密的字节数组 | 否 |


| 参数名 | 描述 | 可否为空 |
| --- | --- | --- |
| string | 要加密的字节数组 | 否 |
| charset | 编码格式,默认UTF-8 | 是 |

返回值类型：String

根据字节数组或字符串进行base64加密，返回String

- 根据访问结果进行base64加密


```
${base64.encode(resp.html)}
```


```
${base64.encode(resp.bytes)}
```
