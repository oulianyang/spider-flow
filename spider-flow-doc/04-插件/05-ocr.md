# ocr


#### 引入ocr插件

![ocr - 图1](https://static.bookstack.cn/projects/spiderflow-0.5.0/dd9d9a67022f84f0cec066bb0d05eb2e.svg)
首先到码云下载插件点击下载到本地并导入到工作空间或安装到maven库


#### 使用方法

在 spider-flow-web pom.xml引用


```
<dependency>
     <groupId>org.spiderflow</groupId>
     <artifactId>spider-flow-ocr</artifactId>
</dependency>
```
